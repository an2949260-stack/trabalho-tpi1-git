\# Trabalho TPI1 - Git



\## Descrição



Este repositório contém o desenvolvimento do trabalho prático da disciplina \*\*Tecnologias para Internet I\*\*, com foco na utilização do sistema de controle de versão Git e da plataforma GitHub.



O objetivo do projeto foi gerenciar versões de um código Java utilizando branches, commits e merges, conforme as instruções fornecidas pelo professor.



\---



\## Objetivo



Aplicar na prática os principais conceitos de Git, incluindo:



\* Controle de versão

\* Criação de branches

\* Commits

\* Push para repositório remoto

\* Merge de branches

\* Resolução de conflitos



\---



\## Tecnologias utilizadas



\* Git

\* GitHub

\* Java

\* Terminal (CLI)



\---



\## Estrutura de Branches



O projeto foi desenvolvido utilizando as seguintes branches:



\* `main` → Branch principal com o resultado final

\* `amissao` → Implementação parcial da calculadora

\* `oresgate` → Outra versão da aplicação

\* `oamanhanuncamorre` → Versão alternativa do código



\---



\## Fluxo de desenvolvimento



1\. Criação do repositório no GitHub

2\. Clonagem do projeto localmente

3\. Criação das branches auxiliares

4\. Adição de código específico em cada branch

5\. Commit e push das alterações

6\. Edição da branch principal (`main`)

7\. Realização dos merges

8\. Resolução de conflitos

9\. Consolidação do código final
